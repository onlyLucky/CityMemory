# 城迹 - 后端接口文档

## 一、文档概述

### 1.1 文档目的
本文档详细描述"城迹"微信小程序后端API接口规范，包括接口地址、请求参数、响应格式及错误码，为前后端开发提供接口对接依据。

### 1.2 接口基础信息

| 项目 | 说明 |
|------|------|
| 基础地址 | https://api.citytrace.com/api/v1 |
| 协议 | HTTPS |
| 数据格式 | JSON |
| 编码 | UTF-8 |
| 认证方式 | Bearer Token (JWT) |

### 1.3 请求头规范

```
Content-Type: application/json
Authorization: Bearer {token}
X-Request-ID: {uuid}
X-Client-Version: 1.0.0
X-Platform: miniprogram
```

### 1.4 响应格式规范

#### 1.4.1 成功响应

```json
{
  "code": 0,
  "message": "success",
  "data": {
  },
  "timestamp": 1711080000
}
```

#### 1.4.2 错误响应

```json
{
  "code": 1001,
  "message": "参数错误",
  "data": null,
  "timestamp": 1711080000
}
```

### 1.5 错误码规范

| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 1001 | 参数错误 |
| 1002 | 未授权 |
| 1003 | 禁止访问 |
| 1004 | 资源不存在 |
| 1005 | 请求方法不允许 |
| 1006 | 请求超时 |
| 2001 | 用户不存在 |
| 2002 | 用户已存在 |
| 2003 | 用户已禁用 |
| 3001 | 门票不足 |
| 3002 | 关卡未解锁 |
| 3003 | 关卡已完成 |
| 3004 | 答题已结束 |
| 4001 | 星星不足 |
| 4002 | 金币不足 |
| 4003 | 道具不足 |
| 4004 | 商品不存在 |
| 5001 | 服务器内部错误 |
| 5002 | 数据库错误 |
| 5003 | 缓存错误 |
| 5004 | 第三方服务错误 |

## 二、用户模块接口

### 2.1 微信登录

**接口描述**：用户通过微信授权登录

**请求信息**：

| 项目 | 说明 |
|------|------|
| URL | /user/login |
| Method | POST |
| 认证 | 否 |

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| code | String | 是 | 微信登录code |

**请求示例**：

```json
{
  "code": "071abc123def456"
}
```

**响应参数**：

| 参数名 | 类型 | 说明 |
|--------|------|------|
| token | String | JWT Token |
| user | Object | 用户信息 |
| user.user_id | String | 用户ID |
| user.nickname | String | 用户昵称 |
| user.avatar | String | 用户头像URL |
| user.province | String | 用户省份 |
| user.total_stars | Decimal | 总星星数 |
| user.level_count | Integer | 已通关卡数 |
| user.coins | Integer | 金币数 |
| user.is_new | Boolean | 是否新用户 |

**响应示例**：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "user": {
      "user_id": "U001",
      "nickname": "用户abc123",
      "avatar": "https://cdn.example.com/default-avatar.png",
      "province": null,
      "total_stars": 0,
      "level_count": 0,
      "coins": 0,
      "is_new": true
    }
  },
  "timestamp": 1711080000
}
```

### 4.2 提交闯关答案

| 项目 | 说明 |
|------|------|
| URL | /answer/submit-adventure |
| Method | POST |
| 认证 | 是 |

| 参数名 | 类型 | 必填 | 说明 |

      "question_id": "Q001",

      "time_used": 4500

      "question_id": "Q002",

| unlock.new_region | String | 新解锁区域 |

  success: (res) => {
    const code = res.code
    // 2. 调用后端登录接口
    request.post('/user/login', { code })
      .then((result) => {
        const { token, user } = result.data
        // 3. 存储Token
        wx.setStorageSync('token', token)
        // 4. 判断是否需要初始化省份
        if (user.is_new || !user.province) {
          wx.redirectTo({ url: '/pages/province-init/province-init' })
        } else {
          wx.switchTab({ url: '/pages/home/home' })
        }
      })
      .catch((error) => {
        console.error('登录失败', error)
      })
  }
})
