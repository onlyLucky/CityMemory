# 城迹 - 代码开发规范文档

## 一、文档概述

### 1.1 文档目的
本文档定义"城迹"微信小程序项目的代码开发规范，包括命名规范、代码风格、目录结构、注释规范等，确保团队代码风格统一，提高代码可读性和可维护性。

### 1.2 适用范围
- 小程序前端代码
- 后端服务代码
- 数据库脚本
- 配置文件

### 1.3 规范原则

| 原则 | 说明 |
|------|------|
| 一致性 | 团队内代码风格保持一致 |
| 可读性 | 代码易于理解和维护 |
| 简洁性 | 避免过度设计，保持代码简洁 |
| 规范性 | 遵循行业最佳实践 |

## 二、通用规范

### 2.1 编码规范

#### 2.1.1 字符编码
- 所有源文件使用 **UTF-8** 编码
- 换行符使用 **LF** (Unix风格)
- 文件末尾保留一个空行

#### 2.1.2 缩进规范
- 使用 **2空格** 缩进
- 禁止使用 Tab 字符
- 每行代码不超过 **100** 字符

#### 2.1.3 注释语言
- 代码注释使用 **中文**
- 变量命名使用 **英文**
- Git提交信息使用 **中文**

### 2.2 命名规范

#### 2.2.1 通用命名规则

| 类型 | 命名风格 | 示例 |
|------|----------|------|
| 文件名 | kebab-case | `user-service.js` |
| 变量名 | camelCase | `userName` |
| 常量名 | UPPER_SNAKE_CASE | `MAX_RETRY_COUNT` |
| 函数名 | camelCase | `getUserInfo` |
| 类名 | PascalCase | `UserService` |
| 数据库表名 | snake_case | `user_level` |
| 数据库字段名 | snake_case | `total_stars` |

#### 2.2.2 命名禁用词

```
data, info, temp, tmp, obj, val, num, str, arr, list
```

应使用更具描述性的名称：
```
userData → userProfile
tempArr → pendingItems
obj → currentUser
```

### 2.3 注释规范

#### 2.3.1 文件头注释

```javascript
/**
 * @file 文件名称
 * @description 文件描述
 * @author 作者
 * @date 创建日期
 * @update 修改记录
 */
```

#### 2.3.2 函数注释

```javascript
/**
 * 函数功能描述
 * @param {Type} paramName 参数说明
 * @returns {Type} 返回值说明
 * @throws {Error} 异常说明
 * @example
 * // 使用示例
 * const result = functionName(param);

  // 实现

```

#### 2.3.3 行内注释

```javascript

      type: Object,
      value: null,
      observer: '_onQuestionChange'
    },
    showAnswer: {
      type: Boolean,
      value: false
    },
    disabled: {
      type: Boolean,
      value: false
    }
  },

  data: {
    selectedOption: null,
    options: []
  },

  lifetimes: {
    attached() {
      this.initComponent()
    }
  },

  methods: {
    initComponent() {
      // 初始化逻辑
    },

    _onQuestionChange(newVal, oldVal) {
      if (newVal) {
        this.setData({
          options: newVal.options,
          selectedOption: null
        })
      }
    },

    handleOptionTap(event) {
      const { option } = event.currentTarget.dataset
      if (this.data.disabled) return

      this.setData({ selectedOption: option.key })
      this.triggerEvent('select', { option })
    }
  }
})
```

### 3.5 工具函数规范

#### 3.5.1 请求封装

```javascript
// utils/request.js

/**
 * @file 请求封装
 * @description 封装微信请求，统一处理错误和Token刷新
 * @author 开发团队
 * @date 2026-03-21
 */

const BASE_URL = 'https://api.citytrace.com/api/v1'
const TOKEN_KEY = 'access_token'

class Request {
  constructor() {
    this.pendingRequests = new Map()
  }

  generateRequestKey(config) {
    const { url, method, data } = config
    return `${method}:${url}:${JSON.stringify(data)}`
  }

  addPendingRequest(config) {
    const key = this.generateRequestKey(config)
    if (this.pendingRequests.has(key)) {
      const cancel = this.pendingRequests.get(key)
      cancel('请求重复，已取消')
    }
    config.cancelToken = new Promise((resolve) => {
      this.pendingRequests.set(key, resolve)
    })
  }

  removePendingRequest(config) {
    const key = this.generateRequestKey(config)
    this.pendingRequests.delete(key)
  }

  getToken() {
    return wx.getStorageSync(TOKEN_KEY)
  }

  setToken(token) {
    wx.setStorageSync(TOKEN_KEY, token)
  }

  clearToken() {
    wx.removeStorageSync(TOKEN_KEY)
  }

  async refreshToken() {
    const refreshToken = wx.getStorageSync('refresh_token')
    if (!refreshToken) {
      throw new Error('无刷新Token')
    }
    const res = await this.request({
      url: '/auth/refresh',
      method: 'POST',
      data: { refreshToken },
      skipAuth: true
    })
    this.setToken(res.data.token)
    return res.data.token
  }

  async request(options) {
    const {
      url,
      method = 'GET',
      data = {},
      header = {},
      skipAuth = false,
      showLoading = true,
      showError = true
    } = options

    if (showLoading) {
      wx.showLoading({ title: '加载中', mask: true })
    }

    const token = this.getToken()
    if (token && !skipAuth) {
      header.Authorization = `Bearer ${token}`
    }

    return new Promise((resolve, reject) => {
      wx.request({
        url: `${BASE_URL}${url}`,
        method,
        data,
        header: {
          'Content-Type': 'application/json',
          ...header
        },
        success: async (res) => {
          if (showLoading) {
            wx.hideLoading()
          }

          if (res.statusCode === 200) {
            if (res.data.code === 0) {
              resolve(res.data)
            } else {
              if (showError) {
                wx.showToast({ title: res.data.message, icon: 'none' })
              }
              reject(res.data)
            }
          } else if (res.statusCode === 401) {
            try {
              const newToken = await this.refreshToken()
              header.Authorization = `Bearer ${newToken}`
              const retryRes = await this.request({ ...options, skipAuth: true })
              resolve(retryRes)
            } catch (error) {
              this.clearToken()
              wx.reLaunch({ url: '/pages/auth/auth' })
              reject(error)
            }
          } else {
            if (showError) {
              wx.showToast({ title: '网络错误', icon: 'none' })
            }
            reject(new Error('网络错误'))
          }
        },
        fail: (error) => {
          if (showLoading) {
            wx.hideLoading()
          }
          if (showError) {
            wx.showToast({ title: '网络错误', icon: 'none' })
          }
          reject(error)
        }
      })
    })
  }

  get(url, data, options = {}) {
    return this.request({ url, method: 'GET', data, ...options })
  }

  post(url, data, options = {}) {
    return this.request({ url, method: 'POST', data, ...options })
  }

  put(url, data, options = {}) {
    return this.request({ url, method: 'PUT', data, ...options })
  }

  delete(url, data, options = {}) {
    return this.request({ url, method: 'DELETE', data, ...options })
  }
}

const request = new Request()

export default request
export { request }
```

#### 3.5.2 存储封装

```javascript
// utils/storage.js

/**
 * @file 存储封装
 * @description 封装本地存储，支持过期时间和加密
 * @author 开发团队
 * @date 2026-03-21
 */

const PREFIX = 'city_trace_'

class Storage {
  constructor() {
    this.prefix = PREFIX
  }

  getKey(key) {
    return `${this.prefix}${key}`
  }

  set(key, value, expireTime = null) {
    const data = {
      value,
      expireTime: expireTime ? Date.now() + expireTime : null
    }
    wx.setStorageSync(this.getKey(key), JSON.stringify(data))
  }

  get(key, defaultValue = null) {
    const data = wx.getStorageSync(this.getKey(key))
    if (!data) return defaultValue

    try {
      const parsed = JSON.parse(data)
      if (parsed.expireTime && Date.now() > parsed.expireTime) {
        this.remove(key)
        return defaultValue
      }
      return parsed.value
    } catch (error) {
      return defaultValue
    }
  }

  remove(key) {
    wx.removeStorageSync(this.getKey(key))
  }

  clear() {
    const res = wx.getStorageInfoSync()
    res.keys.forEach(key => {
      if (key.startsWith(this.prefix)) {
        wx.removeStorageSync(key)
      }
    })
  }

  getInfo() {
    return wx.getStorageInfoSync()
  }
}

const storage = new Storage()

export default storage
export { storage }
```

```javascript
// src/api/user.js

/**
 * @file 用户接口
 * @description 用户相关API路由
 * @author 开发团队
 * @date 2026-03-21
 */

const Router = require('koa-router')
const router = new Router({ prefix: '/api/v1/user' })
const userService = require('../services/user.service')
const { validate } = require('../middlewares/validate')
const { auth } = require('../middlewares/auth')
const { userSchemas } = require('../validators/user')

/**
 * 微信登录
 * @route POST /api/v1/user/login
 * @param {string} code - 微信登录code
 * @returns {object} 用户信息和Token
 */
router.post('/login', 
  validate(userSchemas.login),
  async (ctx) => {
    const { code } = ctx.request.body
    const result = await userService.login(code)
    ctx.body = {
      code: 0,
      message: 'success',
      data: result
    }
  }
)

/**
 * 获取用户信息
 * @route GET /api/v1/user/info
 * @returns {object} 用户信息
 */
router.get('/info', 
  auth(),
  async (ctx) => {
    const userId = ctx.state.user.userId
    const userInfo = await userService.getUserInfo(userId)
    ctx.body = {
      code: 0,
      message: 'success',
      data: userInfo
    }
  }
)

/**
 * 更新用户信息
 * @route PUT /api/v1/user/info
 * @param {object} userInfo - 用户信息
 * @returns {object} 更新后的用户信息
 */
router.put('/info',
  auth(),
  validate(userSchemas.update),
  async (ctx) => {
    const userId = ctx.state.user.userId
    const updateData = ctx.request.body
```

### 4.3 服务层规范

```javascript
// src/services/user.service.js

/**
 * @file 用户服务
 * @description 用户相关业务逻辑
 * @author 开发团队
 * @date 2026-03-21
 */

const UserModel = require('../models/user.model')
const UserLevelModel = require('../models/user_level.model')
const { generateToken, verifyToken } = require('../utils/jwt')
const { redis } = require('../utils/redis')
const { logger } = require('../utils/logger')
const { getWxOpenid } = require('../utils/wechat')

class UserService {
  constructor() {
    this.userModel = new UserModel()
    this.userLevelModel = new UserLevelModel()
    this.cachePrefix = 'user:'
    this.cacheTTL = 24 * 60 * 60
  }

  /**
   * 用户登录
   * @param {string} code - 微信登录code
   * @returns {object} 用户信息和Token
   */
  async login(code) {
    const openid = await getWxOpenid(code)
    if (!openid) {
      throw new Error('获取OpenID失败')
    }

    let user = await this.userModel.findByOpenid(openid)
    
    if (!user) {
      user = await this.userModel.create({
        openid,
        nickname: '用户' + Date.now().toString(36),
        avatar: '',
        total_stars: 0,
        level_count: 0,
        coins: 0
      })
      logger.info('新用户注册', { userId: user.user_id })
    }

    await this.userModel.updateLastLogin(user.user_id)

    const token = generateToken({ userId: user.user_id, openid })
    await redis.setex(
      `${this.cachePrefix}${user.user_id}`,
      this.cacheTTL,
      JSON.stringify(user)
    )

    return {
      token,
      user: this.sanitizeUser(user)
    }
  }

  /**
   * 获取用户信息
   * @param {string} userId - 用户ID
   * @returns {object} 用户信息
   */
  async getUserInfo(userId) {
    const cacheKey = `${this.cachePrefix}${userId}`
    const cachedUser = await redis.get(cacheKey)
    
    if (cachedUser) {
      return this.sanitizeUser(JSON.parse(cachedUser))
    }

    const user = await this.userModel.findById(userId)
    if (!user) {
      throw new Error('用户不存在')
    }

    await redis.setex(cacheKey, this.cacheTTL, JSON.stringify(user))
    return this.sanitizeUser(user)
  }

  /**
   * 更新用户信息
   * @param {string} userId - 用户ID
   * @param {object} updateData - 更新数据
   * @returns {object} 更新后的用户信息
   */
  async updateUserInfo(userId, updateData) {
    const allowedFields = ['nickname', 'avatar', 'province']
    const filteredData = {}
    
    Object.keys(updateData).forEach(key => {
      if (allowedFields.includes(key)) {
        filteredData[key] = updateData[key]
      }
    })

    await this.userModel.update(userId, filteredData)
    
    const cacheKey = `${this.cachePrefix}${userId}`
    await redis.del(cacheKey)

    return this.getUserInfo(userId)
  }

  /**
   * 初始化省份
   * @param {string} userId - 用户ID
   * @param {string} province - 省份名称
   * @param {string} method - 初始化方式
   * @returns {object} 初始化结果
   */
  async initProvince(userId, province, method) {
    const user = await this.userModel.findById(userId)
    if (user.province) {
      throw new Error('已初始化省份')
    }

    await this.userModel.update(userId, { province })

    const level = await this.userLevelModel.create({
      user_id: userId,
      level_id: `province_${province}`,
      status: 1
    })

    const cacheKey = `${this.cachePrefix}${userId}`
    await redis.del(cacheKey)

    logger.info('用户初始化省份', { userId, province, method })

    return {
      province,
      unlockedLevel: level.level_id
    }
  }

  /**
   * 更新用户星级
   * @param {string} userId - 用户ID
   * @param {number} starDiff - 星级增量
   */
  async updateStars(userId, starDiff) {
    if (starDiff <= 0) return

    await this.userModel.incrementStars(userId, starDiff)
    
    const cacheKey = `${this.cachePrefix}${userId}`
    await redis.del(cacheKey)

    logger.info('用户星级更新', { userId, starDiff })
  }

  /**
   * 清理用户敏感信息
   * @param {object} user - 用户对象
   * @returns {object} 清理后的用户信息
   */
  sanitizeUser(user) {
    const { openid, ...safeUser } = user
    return safeUser
  }
}

module.exports = new UserService()
```

```javascript
const { db } = require('../config/database')
const { generateId } = require('../utils/common')

class UserModel {
  constructor() {
    this.tableName = 'user'
  }

  /**
   * 创建用户
   * @param {object} userData - 用户数据
   * @returns {object} 创建的用户
   */
  async create(userData) {
    const userId = generateId('U')
    const [result] = await db(this.tableName).insert({
      user_id: userId,
      ...userData,
      created_at: new Date(),
      updated_at: new Date()
    }).returning('*')
    return result
  }

  /**
   * 根据ID查询用户
   * @param {string} userId - 用户ID
   * @returns {object|null} 用户对象
   */
  async findById(userId) {
    const [user] = await db(this.tableName)
      .where('user_id', userId)
      .where('status', 1)
      .select('*')
    return user || null
  }

  /**
   * 根据OpenID查询用户
   * @param {string} openid - 微信OpenID
   * @returns {object|null} 用户对象
   */
  async findByOpenid(openid) {
    const [user] = await db(this.tableName)
      .where('openid', openid)
      .where('status', 1)
      .select('*')
    return user || null
  }

  /**
   * 更新用户信息
   * @param {string} userId - 用户ID
   * @param {object} updateData - 更新数据
   * @returns {boolean} 是否成功
   */
  async update(userId, updateData) {
    const result = await db(this.tableName)
      .where('user_id', userId)
      .update({
        ...updateData,
        updated_at: new Date()
      })
    return result > 0
  }

  /**
   * 更新最后登录时间
   * @param {string} userId - 用户ID
   */
  async updateLastLogin(userId) {
    await db(this.tableName)
      .where('user_id', userId)
      .update({ last_login_at: new Date() })
  }

  /**
   * 增加星级
   * @param {string} userId - 用户ID
   * @param {number} stars - 星级增量
   */
  async incrementStars(userId, stars) {
    await db(this.tableName)
      .where('user_id', userId)
      .increment('total_stars', stars)
      .increment('level_count', 1)
  }

  /**
   * 获取排行榜用户列表
   * @param {number} limit - 数量限制
   * @returns {array} 用户列表
   */
  async getRankList(limit = 100) {
    return db(this.tableName)
      .where('status', 1)
      .orderBy('total_stars', 'desc')
      .orderBy('level_count', 'desc')
      .orderBy('created_at', 'asc')
      .limit(limit)
      .select('user_id', 'nickname', 'avatar', 'total_stars', 'level_count')
  }
}

module.exports = UserModel
```

```javascript
// src/middlewares/auth.js

/**
 * @file 鉴权中间件
 * @description JWT Token验证
 * @author 开发团队
 * @date 2026-03-21
 */

const { verifyToken } = require('../utils/jwt')
const { redis } = require('../utils/redis')

/**
 * 鉴权中间件
 * @param {object} options - 配置选项
 * @returns {function} Koa中间件
 */
function auth(options = {}) {
  const { required = true } = options

  return async (ctx, next) => {
    const token = ctx.headers.authorization?.replace('Bearer ', '')

    if (!token) {
      if (required) {
        ctx.status = 401
        ctx.body = {
          code: 1002,
          message: '未授权访问'
        }
        return
      }
      await next()
      return
    }

    const isBlacklisted = await redis.get(`token:blacklist:${token}`)
    if (isBlacklisted) {
      ctx.status = 401
      ctx.body = {
        code: 1002,
        message: 'Token已失效'
      }
      return
    }

    try {
      const decoded = verifyToken(token)
      ctx.state.user = decoded
      await next()
    } catch (error) {
      ctx.status = 401
      ctx.body = {
        code: 1002,
        message: 'Token验证失败'
      }
    }
  }
}

module.exports = { auth }
```

### 5.1 表设计规范

#### 5.1.1 表命名规范
- 使用小写字母和下划线
- 使用名词单数形式
- 表名应有业务含义

#### 5.1.2 字段命名规范
- 使用小写字母和下划线
- 主键统一使用 `id`
- 外键使用 `{table}_id` 格式
- 布尔字段使用 `is_{action}` 格式
- 时间字段使用 `{action}_at` 格式

#### 5.1.3 必备字段

每个表必须包含以下字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT UNSIGNED | 自增主键 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 5.2 索引规范

#### 5.2.1 索引命名

| 索引类型 | 命名规则 | 示例 |
|----------|----------|------|
| 主键 | pk_{table} | pk_user |
| 唯一索引 | uk_{column} | uk_user_id |
| 普通索引 | idx_{column} | idx_total_stars |
| 组合索引 | idx_{col1}_{col2} | idx_user_level |

#### 5.2.2 索引原则
- 为WHERE条件字段创建索引
- 为JOIN关联字段创建索引
- 为ORDER BY排序字段创建索引
- 避免在低选择性字段创建索引
- 组合索引遵循最左前缀原则
