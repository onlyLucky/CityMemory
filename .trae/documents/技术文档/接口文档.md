# 城迹服务端 - API接口文档

## 一、文档概述

### 1.1 文档说明
本文档定义"城迹"项目服务端API接口规范，包括小程序端和后台管理系统端的所有接口。服务端统一为小程序和后台管理系统提供API服务。

### 1.2 基础信息

| 项目 | 内容 |
|------|------|
| API基础路径 | `/api/v1` |
| 协议 | HTTPS |
| 数据格式 | JSON |
| 字符编码 | UTF-8 |

### 1.3 通用响应格式

#### 成功响应
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

#### 失败响应
```json
{
  "code": 400,
  "message": "参数错误",
  "data": null
}
```

### 1.4 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未授权或token无效 |
| 403 | 已登录无权限 |
| 404 | 资源不存在 |
| 405 | 方法不存在 |
| 409 | 数据冲突，账号已存在 |
| 422 | 请求参数验证失败 |
| 429 | 超出请求频率 |
| 500 | 服务器内部错误 |
| 502 | 网关错误 |
| 503 | 服务不可用 |
| 504 | 网关超时 |

#### 2.2.4 获取城市列表

**接口地址**：`GET /level/cities`

**接口描述**：获取城市数据，用于本地生成题目。支持按区域、国家、省份筛选，支持分页和批量获取

**请求头**：
```
Authorization: Bearer {token}
```

**请求参数**：
```
?regionId=区域ID&countryId=国家ID&provinceId=省份ID&page=1&pageSize=100&count=50
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| regionId | string | 否 | 区域ID，不传则获取全部 |
| countryId | string | 否 | 国家ID，不传则获取全部 |
| provinceId | string | 否 | 省份ID，不传则获取全部 |
| page | number | 否 | 页码，默认1 |
| pageSize | number | 否 | 每页数量，默认100，最大500 |
| count | number | 否 | 随机获取数量，设置后忽略分页，随机返回指定数量 |

**响应数据**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "id": "城市ID",
        "name": "南京市",
        "provinceId": "省份ID",
        "provinceName": "江苏省",
        "countryId": "国家ID",
        "countryName": "中国",
        "regionId": "区域ID",
        "regionName": "亚洲",
        "ancientName": "金陵",
        "latitude": 32.0603,

        "isCapital": true,
```

**接口地址**：`POST /level/answer/fail`

**接口描述**：答题失败时记录题目信息，用于统计和优化题库

**请求头**：
```
Authorization: Bearer {token}
```

**请求参数**：
```json
{
  "sessionId": "会话ID",
  "levelId": "关卡ID",
  "questionType": 1,
  "questionContent": "南京市属于哪个省份？",
  "cityId": "城市ID",
  "cityName": "南京市",
  "provinceId": "省份ID",
  "provinceName": "江苏省",
  "userAnswer": "B",
  "correctAnswer": "A",
  "options": [
    { "key": "A", "value": "江苏省" },
    { "key": "B", "value": "浙江省" },
    { "key": "C", "value": "安徽省" },
    { "key": "D", "value": "上海市" }
  ],
  "timeSpent": 5
}
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| sessionId | string | 是 | 会话ID |
| levelId | string | 是 | 关卡ID |
| questionType | number | 是 | 题目类型：1城市猜省份/2城市猜国家/3古称猜今名/4国旗猜国家 |
| questionContent | string | 是 | 题目内容 |
| cityId | string | 是 | 城市ID |
| cityName | string | 是 | 城市名称 |
| provinceId | string | 否 | 省份ID（题目类型1时必填） |
| provinceName | string | 否 | 省份名称 |
| userAnswer | string | 是 | 用户答案 |
| correctAnswer | string | 是 | 正确答案 |
| options | array | 是 | 选项列表 |
| timeSpent | number | 否 | 答题耗时（秒） |

**响应数据**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "recordId": "记录ID",
    "recorded": true
  }
}
```
