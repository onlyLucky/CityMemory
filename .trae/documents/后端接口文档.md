# 城迹运营后台 - 后端接口文档

## 一、文档概述

### 1.1 文档目的
本文档定义"城迹"运营后台管理系统的API接口规范，包括接口地址、请求参数、响应格式、错误码等，为前端开发团队和测试团队提供接口对接依据。

### 1.2 接口规范

#### 1.2.1 基础URL

| 环境 | URL |
|------|-----|
| 开发环境 | `http://localhost:3000/api/v1` |
| 测试环境 | `https://test-api.citytrace.com/api/v1` |
| 生产环境 | `https://api.citytrace.com/api/v1` |

#### 1.2.2 请求头

| 请求头 | 必填 | 说明 |
|--------|------|------|
| Authorization | 是 | Bearer Token认证 |
| Content-Type | 是 | application/json |
| X-Request-Id | 否 | 请求追踪ID |

#### 1.2.3 响应格式

**成功响应**
```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

**分页响应**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "list": [],
    "total": 100,
    "page": 1,
    "pageSize": 20
  }
}
```

**错误响应**
```json
{
  "code": 10001,
  "message": "参数错误",
  "data": null
}
```

#### 1.2.4 错误码定义

| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 10001 | 参数错误 |
| 10002 | 数据不存在 |
| 10003 | 数据已存在 |
| 10004 | 操作失败 |
| 20001 | 未登录 |
| 20002 | 登录已过期 |
| 20003 | 账号或密码错误 |
| 20004 | 账号已被锁定 |
| 20005 | 账号已被禁用 |
| 30001 | 无权限访问 |
| 40001 | 服务器内部错误 |

## 二、认证接口

### 2.1 管理员登录

**接口地址**：`POST /admin/auth/login`

**接口描述**：管理员账号密码登录

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | string | 是 | 登录账号 |
| password | string | 是 | 登录密码 |
| rememberMe | boolean | 否 | 记住登录（7天有效） |

**请求示例**：
```json
{
  "username": "admin",
  "password": "123456",
  "rememberMe": false
}
```

**响应参数**：

| 参数名 | 类型 | 说明 |
|--------|------|------|
| token | string | 访问令牌 |
| refreshToken | string | 刷新令牌 |
| expiresIn | number | 过期时间（秒） |
| userInfo | object | 用户信息 |

**响应示例**：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expiresIn": 7200,
    "userInfo": {
      "id": "ADM001",
      "username": "admin",
      "realName": "管理员",
      "roleId": "ROLE001",
      "roleName": "超级管理员"
    }
  }
}
```

### 2.2 刷新Token

**接口地址**：`POST /admin/auth/refresh`

**接口描述**：使用refreshToken刷新访问令牌

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| refreshToken | string | 是 | 刷新令牌 |

**响应参数**：

| 参数名 | 类型 | 说明 |
|--------|------|------|
| token | string | 新的访问令牌 |
| expiresIn | number | 过期时间（秒） |

### 2.3 退出登录

**接口地址**：`POST /admin/auth/logout`

**接口描述**：管理员退出登录

**请求头**：需要Authorization

**响应示例**：
