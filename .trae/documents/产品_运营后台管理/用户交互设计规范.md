# 城迹运营后台 - 用户交互设计规范

## 一、文档概述

### 1.1 文档目的
本文档定义"城迹"运营后台管理系统的交互设计规范，包括页面结构、操作流程、交互逻辑及体验原则，为UI设计师和开发团队提供设计依据。

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| 高效操作 | 减少操作步骤，支持批量操作和快捷键 |
| 数据清晰 | 数据展示直观，图表可视化 |
| 安全可靠 | 敏感操作二次确认，操作可追溯 |
| 响应及时 | 操作反馈明确，加载状态清晰 |

### 1.3 设计规范

#### 1.3.1 色彩规范

| 色彩类型 | 色值 | 应用场景 |
|----------|------|----------|
| 主色调 | #1890FF | 主按钮、导航栏、链接 |
| 成功色 | #52C41A | 成功状态、启用状态 |
| 警告色 | #FAAD14 | 警告提示、待处理状态 |
| 错误色 | #F5222D | 错误提示、删除按钮 |
| 背景色 | #F0F2F5 | 页面背景 |
| 卡片背景 | #FFFFFF | 卡片、表格背景 |
| 文字主色 | #262626 | 主要文字 |
| 文字辅色 | #8C8C8C | 次要文字、说明文字 |
| 边框色 | #E8E8E8 | 边框、分割线 |

#### 1.3.2 字体规范

| 字体类型 | 字号 | 字重 | 应用场景 |
|----------|------|------|----------|
| 大标题 | 20px | Medium | 页面标题 |
| 标题 | 16px | Medium | 模块标题 |
| 正文 | 14px | Regular | 正文内容 |
| 辅助文字 | 12px | Regular | 说明文字、标签 |

#### 1.3.3 间距规范

| 间距类型 | 数值 | 应用场景 |
|----------|------|----------|
| 页面边距 | 24px | 页面左右边距 |
| 模块间距 | 16px | 模块之间间距 |
| 元素间距 | 8px | 同一模块内元素间距 |
| 表格行高 | 54px | 表格行高度 |

#### 交互逻辑

```
进入登录页
  ↓
输入账号密码
  ↓
点击"登录"
  ↓
验证账号密码
  ├─ 成功 → 记录登录日志 → 跳转数据看板
  └─ 失败 → 显示错误提示 → 错误次数+1
  ↓
错误次数 ≥ 5 → 锁定账号30分钟
```

#### 交互细节

| 操作 | 反馈 |
|------|------|
| 点击登录 | 按钮loading + 禁用 |
| 登录成功 | Toast"登录成功" → 跳转 |
| 登录失败 | Toast"账号或密码错误" |
| 账号锁定 | Toast"账号已锁定，请30分钟后重试" |

## 四、通用交互规范

### 4.1 表格交互

| 操作 | 效果 |
|------|------|
| 点击表头 | 按该列排序 |
| 悬停行 | 行背景高亮 |
| 勾选行 | 批量操作按钮激活 |
| 双击行 | 查看详情（如有） |

### 4.2 表单交互

| 场景 | 交互方式 |
|------|----------|
| 必填项未填 | 提交时标红提示 |
| 格式错误 | 失焦时校验提示 |
| 保存成功 | Toast提示 + 返回列表 |
| 保存失败 | Toast提示错误信息 |

### 4.3 确认弹窗

**删除确认**：

```
┌─────────────────────────────────┐
│           ⚠️ 确认删除            │
│                                 │
│  确定要删除该记录吗？           │
│  此操作不可恢复                 │
│                                 │
│      [取消]        [确定]       │
└─────────────────────────────────┘
```

### 4.4 加载状态

| 场景 | 加载方式 |
|------|----------|
| 页面加载 | 全屏loading |
| 表格加载 | 表格内loading |
| 按钮提交 | 按钮内loading + 禁用 |

### 4.5 空状态

| 场景 | 空状态设计 |
|------|------------|
| 列表无数据 | 图标 + "暂无数据" |
| 搜索无结果 | 图标 + "未找到匹配结果" |
| 网络错误 | 图标 + "网络异常" + 重试按钮 |
