# 城迹运营后台 - 系统架构设计文档

## 一、文档概述

### 1.1 文档目的
本文档详细描述"城迹"运营后台管理系统的系统架构设计，包括整体架构、模块划分、数据流转、技术架构分层及关键技术实现方案，为开发团队提供架构层面的实施指导。

### 1.2 架构设计原则

| 原则 | 说明 |
|------|------|
| 高可用 | 单点故障不影响整体服务，支持故障转移 |
| 可扩展 | 模块化设计，支持水平扩展 |
| 高性能 | 缓存优先，减少数据库压力 |
| 安全性 | 数据加密，接口鉴权，防攻击 |
| 简洁性 | 避免过度设计，保持架构清晰 |
| 复用性 | 与小程序共享后端服务，降低维护成本 |

### 2.2 架构分层说明

| 层级 | 职责 | 技术实现 |
|------|------|----------|
| 用户层 | 后台管理前端，负责用户交互和业务展示 | Vue 3 + Element Plus |
| 接入层 | SSL终止、负载均衡、静态资源服务 | Nginx |
| 网关层 | 身份认证、限流、日志、错误处理 | Koa中间件 |
| 服务层 | 核心业务逻辑处理 | Node.js服务 |
| 数据层 | 持久化数据存储 | MySQL |
| 缓存层 | 热点数据缓存、会话管理 | Redis |
| 文档层 | 题库、城市、国家等文档数据 | MongoDB |
| 存储层 | 图片、静态资源存储 | 腾讯云COS + CDN |

### 7.1 JWT认证实现

```typescript
import jwt from 'jsonwebtoken'
import bcrypt from 'bcrypt'
import { redis } from '@/utils/redis'

interface TokenPayload {
  adminId: string
  username: string
  roleId: string
}

class AuthService {
  private readonly JWT_SECRET = process.env.JWT_SECRET!
  private readonly JWT_EXPIRES_IN = '2h'
  private readonly REFRESH_EXPIRES_IN = '7d'

  async login(username: string, password: string) {
    const admin = await AdminModel.findByUsername(username)
    
    if (!admin) {
      throw new Error('账号或密码错误')
    }

    if (admin.status === 0) {
      throw new Error('账号已被禁用')
    }

    if (admin.lockUntil && new Date() < new Date(admin.lockUntil)) {
      throw new Error('账号已被锁定，请稍后重试')
    }

    const isValid = await bcrypt.compare(password, admin.password)
    
    if (!isValid) {
      await this.handleLoginError(username)
      throw new Error('账号或密码错误')
    }

    const token = this.generateToken(admin)
    const refreshToken = this.generateRefreshToken(admin)

    await redis.setex(`admin:token:${admin.id}`, 7200, token)
    await redis.del(`login:error:${username}`)

    return {
      token,
      refreshToken,
      expiresIn: 7200,
      userInfo: {
        id: admin.id,
        username: admin.username,
        realName: admin.realName,
        roleId: admin.roleId
      }
    }
  }

  private generateToken(admin: Admin): string {
    const payload: TokenPayload = {
      adminId: admin.id,
      username: admin.username,
      roleId: admin.roleId
    }
    return jwt.sign(payload, this.JWT_SECRET, { expiresIn: this.JWT_EXPIRES_IN })
  }

  private generateRefreshToken(admin: Admin): string {
    return jwt.sign(
      { adminId: admin.id, type: 'refresh' },
      this.JWT_SECRET,
      { expiresIn: this.REFRESH_EXPIRES_IN }
    )
  }

  async verifyToken(token: string): Promise<TokenPayload> {
    const decoded = jwt.verify(token, this.JWT_SECRET) as TokenPayload
    const cachedToken = await redis.get(`admin:token:${decoded.adminId}`)
    
    if (!cachedToken || cachedToken !== token) {
      throw new Error('Token已失效')
    }

    return decoded
  }
}
```

### 7.2 权限中间件实现

```typescript
import { Context, Next } from 'koa'
import { AuthService } from '@/services/admin/auth.service'

const authService = new AuthService()

export function permissionMiddleware(requiredPermission: string) {
  return async (ctx: Context, next: Next) => {
    const token = ctx.headers.authorization?.replace('Bearer ', '')

    if (!token) {
      ctx.status = 401
      ctx.body = { code: 20001, message: '未登录', data: null }
      return
    }

    try {
      const payload = await authService.verifyToken(token)
      ctx.state.user = payload

      const permissions = await authService.getUserPermissions(payload.adminId)

      if (!permissions.includes('*') && !permissions.includes(requiredPermission)) {
        ctx.status = 403
        ctx.body = { code: 30001, message: '无权限访问', data: null }
        return
      }

      await next()
    } catch (error) {
      ctx.status = 401
      ctx.body = { code: 20002, message: '登录已过期', data: null }
    }
  }
}
```

```typescript

    const result = {

    }

    for (let i = 0; i < rows.length; i++) {

      const errors: string[] = []

      const questionType = row[0]
      const questionContent = row[1]
      const optionA = row[3]

      if (!region) errors.push('区域不能为空')

      if (errors.length > 0) {
        result.invalid.push({ row: i + 2, errors })
      } else {
        result.valid.push({
          questionType,
          questionContent,
          options: [
            { key: 'A', value: optionA },
            { key: 'B', value: optionB },
            { key: 'C', value: optionC },
            { key: 'D', value: optionD }
          ],
          correctAnswer,
          difficulty,
          region,
          status: 2
        })
      }
    }

    return result
  }

  async createImportTask(adminId: string, filePath: string) {
    const taskId = generateId()
    
    await this.importQueue.add('import', {
      taskId,
      adminId,
      filePath
    })

    return taskId
  }

  async processImport(task: ImportTask) {
    const { taskId, adminId, filePath } = task
    
    const parsed = await this.parseExcel(filePath)
    
    let successCount = 0
    let failCount = 0

    for (const question of parsed.valid) {
      try {
        await QuestionModel.create({
          ...question,
          createdBy: adminId
        })
        successCount++
      } catch (error) {
        failCount++
      }
    }

    return {
      successCount,
      failCount,
      invalidCount: parsed.invalid.length
    }
  }
}
```

### 7.4 数据看板统计实现

```typescript
import { redis } from '@/utils/redis'

class DashboardService {
  async getOverview(date: string) {
    const cacheKey = `dashboard:overview:${date}`
    
    const cached = await redis.get(cacheKey)
    if (cached) {
      return JSON.parse(cached)
    }

    const [
      todayNewUsers,
      totalUsers,
      todayActiveUsers,
      todayAnswerCount,
      day1Retention,
      day7Retention
    ] = await Promise.all([
      this.getTodayNewUsers(date),
      this.getTotalUsers(),
      this.getTodayActiveUsers(date),
      this.getTodayAnswerCount(date),
      this.getDay1Retention(date),
      this.getDay7Retention(date)
    ])

    const result = {
      todayNewUsers,
      totalUsers,
      todayActiveUsers,
      todayAnswerCount,
      day1Retention,
      day7Retention
    }

    await redis.setex(cacheKey, 300, JSON.stringify(result))

    return result
  }

  private async getTodayNewUsers(date: string): Promise<number> {
    const result = await UserModel.count({
      where: {
        createdAt: {
          [Op.gte]: `${date} 00:00:00`,
          [Op.lt]: `${date} 23:59:59`
        }
      }
    })
    return result
  }

  private async getDay1Retention(date: string): Promise<number> {
    const yesterday = dayjs(date).subtract(1, 'day').format('YYYY-MM-DD')
    
    const newUsersYesterday = await UserModel.count({
      where: {
        createdAt: {
          [Op.gte]: `${yesterday} 00:00:00`,
          [Op.lt]: `${yesterday} 23:59:59`
        }
      }
    })

    if (newUsersYesterday === 0) return 0

    const retainedUsers = await UserModel.count({
      where: {
        createdAt: {
          [Op.gte]: `${yesterday} 00:00:00`,
          [Op.lt]: `${yesterday} 23:59:59`
        },
        lastLoginAt: {
          [Op.gte]: `${date} 00:00:00`
        }
      }
    })

    return Number((retainedUsers / newUsersYesterday * 100).toFixed(1))
  }
}
```
